Detailed Description

Handles ROOT output creation and management for the simulation.

This class manages the ROOT file output, creating a TTree structure to store simulation results including:

  • Energy deposits in liquid xenon
  • Neutron track lengths
  • Neutron counting

Creates and configures the ROOT output:

  1. Creates a new ROOT file 'neutron_data.root'
  2. Creates a TTree 'neutronTree'
  3. Sets up branches for:
    • Energy deposits (Double_t)
    • Track lengths (Double_t)
    • Neutron counts (Int_t)
